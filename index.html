Texto y formato en línea

Un párrafo se escribe con <p>. Dentro puedes resaltar con <strong>, enfatizar con <em>, resaltar como marcador con <mark>, o citar código con <code>.

Esta es una cita con <blockquote>, ideal para resaltar frases de otras personas.

Usa enlaces <a> para conectar a otras páginas. El atributo target="_blank" abre en otra pestaña.

Los elementos en línea como <span> permiten aplicar estilo sin romper el flujo del texto.

Formularios

Estructura de página

Las etiquetas semánticas como <header>, <nav>, <main>, <section>, <article>, <aside> y <footer> ayudan a describir el propósito de cada bloque.

Artículo destacado

Un <article> agrupa contenido independiente, como una noticia o una entrada de blog.

Barra lateral

Un <aside> contiene información complementaria, como enlaces o anuncios.

Un <div> agrupa bloques y es útil para maquetar y aplicar estilos.
Combínalo con clases o identificadores para controlar su apariencia.
